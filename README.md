# Zentra Funcs

We list the proposed Zentra functions.

  * [committee_init](funcs/zip1.py#L1) the builtin func to init the committee who can approve the funcs.
  * [committee_add_member](funcs/zip1.py#L11) the func to add committee member.
  * [committee_remove_member](funcs/zip1.py#L34) the func to remove committee member.
  * [function_proposal](funcs/zip2.py#L26) the func to proposal new func. It is builtin and will be override by deploy.
  * [function_vote](funcs/zip2.py#L53) the func to vote new func. It is builtin and will be override by deploy.
