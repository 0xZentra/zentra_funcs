# Zentra Funcs


We list the proposed Zentra functions by ZIPs.

*   **committee_init**: The built-in function to initialize the committee that can approve functions.
*   [ZIP1](https://github.com/0xZentra/zentra_funcs/blob/cba856ee673d03a6b66b2720deb8d45c0eb2335e/funcs/zip1.py): Functions for committee management, including adding and removing members.
*   [ZIP2](https://github.com/0xZentra/zentra_funcs/blob/cba856ee673d03a6b66b2720deb8d45c0eb2335e/funcs/zip2.py): Functions for proposing and voting on new functions to be added to the system.
*   [ZIP3](https://github.com/0xZentra/zentra_funcs/blob/cba856ee673d03a6b66b2720deb8d45c0eb2335e/funcs/zip3.py): Functions for creating and managing assets, including updating ownership and functions.
*   [ZIP20](https://github.com/0xZentra/zentra_funcs/blob/3ab9b48aac2189831d71878ef121c2cf41aadd44/funcs/zip20.py): Functions for creating and managing tokens, including minting, burning, and transferring.
*   [ZIP21](https://github.com/0xZentra/zentra_funcs/blob/cba856ee673d03a6b66b2720deb8d45c0eb2335e/funcs/zip21.py): Functions for bridging assets from other blockchains, including handling incoming and outgoing transfers.
*   [ZIP22](funcs/zip22.py): A decentralized orderbook exchange for trading assets, with support for limit and market orders.
